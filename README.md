# @orchestra-mcp/theme

Color themes and CSS variables for Orchestra MCP — Tailwind CSS v4

Part of the [Orchestra MCP](https://orchestra-mcp.dev) ecosystem.

## Install

```bash
npm install @orchestra-mcp/theme
# or
pnpm add @orchestra-mcp/theme
```

## Usage

```tsx
import { } from '@orchestra-mcp/theme'
```

## Requirements

- React >= 18.0.0
